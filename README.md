### Featured Projects

**Travel to Spain** — React + Vite + Tailwind  
Real Spanish Tourism API (Dataestur) integration with interactive data visualization and per-destination color themes.  
→ [Live Demo](https://travel-to-spain.vercel.app/) | [Repository](https://github.com/MarcosJVPR/TravelToSpain)

**Hadas Vs Ogros** — Unreal Engine 5 (Blueprints)  
Asymmetric 3D RTS in development. The Sylvan Court vs. The Iron-Crag Clans with era progression, dynamic biomes, and faction-specific visuals. Solo developer, Blueprints-only architecture.
